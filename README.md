# rating-calculator

Rating calculators written in Javascript for Ongeki.

## Usage

```
# Ongeki rating calculator

```

## Installation

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.2.15. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
